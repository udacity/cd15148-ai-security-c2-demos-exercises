# Downloaded Demo Files

This folder is for model files that are downloaded before running the notebook.

The expected model location is:

```text
downloads/gtsrb_model/
```

Run the download script from the demo root:

```bash
bash scripts/download_gtsrb_model.sh
```

The script downloads the Hugging Face model files for `kelvinandreas/vit-traffic-sign-GTSRB`, a MIT-licensed Vision Transformer fine-tuned for 43-class GTSRB traffic sign classification.
