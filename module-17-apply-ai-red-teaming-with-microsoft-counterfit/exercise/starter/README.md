# Starter: Counterfit Drone Detection Assessment

Complete the Counterfit workflow for the drone detection target.

Main files:

- `notebooks/counterfit_drone_detection_assessment.ipynb`
- `targets/drone_detection_target.py`
- `src/drone_counterfit_utils.py`
- `configs/counterfit_attack_plan.json`
- `docs/assessment_report_template.md`

The starter includes TODOs in the notebook and target class. The target must expose a valid Counterfit `CFTarget` with `load(...)` and `predict(...)` behavior.
