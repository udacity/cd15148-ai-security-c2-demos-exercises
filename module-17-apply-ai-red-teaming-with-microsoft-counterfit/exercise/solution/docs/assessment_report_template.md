## Operational Impact

Successful evasion against the positive `drone` class may reduce alert reliability and create false negatives in restricted airspace monitoring. A technically successful attack should be prioritized based on how often it suppresses drone detections, how visible the perturbation is, and how feasible the input manipulation is in the camera pipeline.

## Mitigations

- Add adversarial robustness testing to model release gates.
- Monitor class-specific confidence degradation and false-negative spikes.
- Use ensemble or temporal validation across multiple frames.
- Add camera and input integrity checks.
- Retrain or fine-tune with adversarial examples for high-risk classes.
