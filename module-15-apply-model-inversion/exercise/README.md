# Exercise: Build a Model Inversion Privacy Assessment Workflow for a Medical Diagnosis Model

Estimated time: 45 minutes

Learners implement a model inversion privacy assessment workflow for a brain tumor MRI screening classifier using the Ultralytics brain tumor dataset. The exercise includes starter infrastructure, a completed solution, and generated model/data artifacts.

## Folders

- `starter/`: learner-facing scaffold with TODOs.
- `solution/`: completed reference implementation.

## Deliverables

Students submit a completed notebook, reconstructed feature outputs, quantitative leakage comparisons across at least three output configurations, visual comparisons, and a short privacy assessment report.
