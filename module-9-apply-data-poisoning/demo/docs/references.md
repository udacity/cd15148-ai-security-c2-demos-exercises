## Classroom Runtime Note

The notebook trains ResNet-18 on a 5,000-image CIFAR-10 subset for a small number of epochs so the demo can run in a classroom setting. The resulting clean accuracy is not intended to represent a production-quality CIFAR-10 model. The teaching objective is the contrast between normal clean validation behavior and targeted triggered behavior.
